> rektradar sdk demo

Everything on this page is the official @mik3fly-lab/rektradar-sdk running in your browser, on the free tier - no API key, no signup.

free tier: targeted lookups are real-time; the live flow is ~10 min delayed.

// the entire integration

npm install @mik3fly-lab/rektradar-sdk

import { RektRadar, connectStream } from "@mik3fly-lab/rektradar-sdk";

const rr = new RektRadar();                       // free tier - no API key

const v = await rr.token("0x...");                // real-time risk verdict
const { rugs } = await rr.rugs({ since: "30d" }); // recent rugs (~10 min delayed on free)
const { trends } = await rr.trends({ period: "7d", granularity: "daily" });

connectStream({                                   // live feed over WebSocket
  events: ["new_token", "imminent_rug", "rug"],
  onMessage: (e) => console.log(e.type, e.data),
});
